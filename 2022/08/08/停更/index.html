博客停止更新声明

看到这些文字的朋友们，感谢你们来访我的个人主页！但是很抱歉，我暂时不打算继续更新它们了，也不知道什么时候会再次回来——甚至可能永远不回来了。这篇文章将简单地谈一谈我停更的原因。

当初打算开一个博客，可能是单纯地觉得，有一个属于我的网站是一件很酷的事情。同时也是想简单接触一些网络知识，于是就在多方帮助之下（感谢 rcy 讲授的 Hexo 课程，感谢 lr 和 wl 在我遇到困难时给予我的技术指导），使用 Hexo + Github Pages 搭建了这么一个小小的个人主页。

到现在为止，博客一共更新了8篇文章，有一些生活的记录，也有一些计算机技术的介绍，和朋友们的博客里的内容也基本相似。进入大学以来，在互联网上发表文字已经成了多数人的生活日常。当然，不同于常用的 QQ空间、微信朋友圈、小红书或者微博，大家普遍选择了一些更加“高级”的方式——例如个人公众号，B站，或者个人网站。我承认，比起那些“傻瓜式”的发文手段，这些的确更讲求个人能力。从这个角度上说，的确是更高级一些。但其实，各种手段的目的都只不过是为了分享、记录生活（技术也算是生活吧？），既然如此，它们又有什么本质区别呢？又何谈高级与不高级呢？

何况互联网是没有记忆的。或许从哪一天开始，腾讯不再运营了，翻墙也做不到了，到那个时候，我们尝试记录和分享的文字也就这么消失了——不再有人能看见，不再有人能进入我们的内心。伴随着一起消失的，恐怕也是我们对互联网的全部热情和信念。

另外，我发表这些文字，到底是为了给谁看呢？从搭建博客开始，我就一直在思考这个问题。后来才想明白，真正在意它的，其实只有我自己。然而一旦这个本该是私密性的世界被公开给所有人，我们就不得不谨言慎行，许多真正的心里话也被隐藏在了这些文字背后。那么，记录生活的意义又何在呢？

我开始为此感到深深的迷茫。

所以啊，到此为止吧——或许有一天，我也会再次回来。
