各种STL容器的迭代器属性如下：

  • Random Access: vector, queue, array*
  • Bidirectional: map, set, list
  • Forward: unordered_map, unordered_set, forward_list

Random Access Iterator是最厉害的，可以理解为：计算任意两个迭代器之间的距离只需要用到$O(1)$的时间——例如，对vector这种call by rank的容器，只需计算两个元素的索引之差即可。但另外两种迭代器是类似“链表”的串接形式，想要计算距离，只能顺着链表节点慢慢地爬，时间是$O(n)$的。

其中，map和set是双向的（意思是支持自增++和自减–），是因为它们的实现方式是红黑树，支持节点向前、向后索引；而unordered_map/set只支持自增，不支持自减，因为它们的实现方式是哈希表。

这就是模板元编程的动机。

Meta-function

Definition: A meta-function is a “function” that operates on some types/values (“parameters”) and outputs some types/values (“return values”).

More: A meta-functin is a struct that has public member types/fields which depend on what the template types/values are instantiated with.

也就是说，我们实现my_distance这种类似函数的语句的方法是：定义一个结构体。

如何修改这个缺陷呢？

constexpr

C++17新增了特性：if constexpr。You can calculate the boolean at compile-time. Replace the entire if/else with the code that will actually be run. （注意：一定要C++17！）意思是：编译器并不会预先就把两个分支都编译了，而是会先计算if后面的条件判断值，如果它是false，就不执行if分支了。
